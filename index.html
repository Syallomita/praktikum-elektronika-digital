Gerbang AND menghasilkan 1 hanya jika semua inputnya 1. Y = A AND B. Tabel kebenaran: 0 AND 0 = 0, 0 AND 1 = 0, 1 AND 0 = 0, 1 AND 1 = 1

Gerbang OR

Output Y: 0
Penjelasan
Gerbang OR menghasilkan 1 jika salah satu atau semua inputnya 1. Y = A OR B. Tabel kebenaran: 0 OR 0 = 0, 0 OR 1 = 1, 1 OR 0 = 1, 1 OR 1 = 1

Gerbang NOT

Output Y: 1
Penjelasan
Gerbang NOT membalik inputnya. Y = NOT A. Jika A=0 maka Y=1, dan sebaliknya. Tabel kebenaran: NOT 0 = 1, NOT 1 = 0

Gerbang XOR

Output Y: 0
Penjelasan
Gerbang XOR (Exclusive OR) menghasilkan 1 jika inputnya berbeda. Y = A XOR B. Tabel kebenaran: 0 XOR 0 = 0, 0 XOR 1 = 1, 1 XOR 0 = 1, 1 XOR 1 = 0

Gerbang NAND

Output Y: 1
Penjelasan
Gerbang NAND adalah kebalikan dari AND. Y = NOT (A AND B). Tabel kebenaran: 0 NAND 0 = 1, 0 NAND 1 = 1, 1 NAND 0 = 1, 1 NAND 1 = 0

Gerbang NOR

Output Y: 1
Penjelasan
Gerbang NOR adalah kebalikan dari OR. Y = NOT (A OR B). Tabel kebenaran: 0 NOR 0 = 1, 0 NOR 1 = 0, 1 NOR 0 = 0, 1 NOR 1 = 0

Info Gerbang Logika

Gerbang AND

Menghasilkan 1 hanya jika semua inputnya 1. Digunakan dalam operasi yang membutuhkan semua kondisi terpenuhi.

Gerbang OR

Menghasilkan 1 jika minimal satu input adalah 1. Berguna untuk situasi dimana beberapa kondisi bisa memicu aksi yang sama.

Gerbang NOT

Membalik input, mengubah 0 menjadi 1 dan sebaliknya. Digunakan untuk inversi logika.

Gerbang XOR

Menghasilkan 1 jika input berbeda. Berguna dalam operasi perbandingan dan deteksi perubahan.

Gerbang NAND

Universal, dapat digunakan untuk membuat semua gerbang lain. Merupakan kombinasi AND dan NOT.

Gerbang NOR

Juga universal seperti NAND. Merupakan kombinasi OR dan NOT.
